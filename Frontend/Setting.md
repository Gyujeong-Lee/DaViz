# Setting

```bash
# 라이브러리 설치
$ npm install
# 실행
$ npm start
```



- vscode extension 설치
  - Prettier - Code formatter
  - ESLint

- settings.json 설정 : js 코드 자동 정렬

```json
    "[javascript]": {
        //default formatter
        "editor.defaultFormatter": "esbenp.prettier-vscode",
        "editor.formatOnSave": true
    },
    "terminal.integrated.defaultProfile.windows": "Git Bash",
    "editor.formatOnSave": true,
```
