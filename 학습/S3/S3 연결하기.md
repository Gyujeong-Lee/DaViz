## 2. Django와 연결

### 1) django_storage 

- S3와 Django 연결, 파일 시스템 관련 메서드를 정의해둔 패키지

- 설치

  ```
  pip install django_storages
  ```

  - `installed_apps` 추가

### 2) boto3 

- S3 통신을 위한 패키지
  - 조작 API 
  - request를 이용하여 요청 보낼 필요 없음
  - Method
    - delete
    - ...

- 설치 

  ```
  pip install boto3
  import boto3
  ```
